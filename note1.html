Google Forms

  • Buka drive.google.com & buat folder baru (contoh: "Sijil_Bulk_Output")
  • Buat Google Docs sebagai template sijil. Guna placeholder: {{Nama}} , {{Program}} , {{Tarikh}}
  • Dapatkan ID Folder & ID Docs Template dari URL (lihat contoh dalam kod).
ID contoh: 1VQaMPsO-QJRN99pWcQu9rGucX2ajI0Zg
2

Sediakan Borang + Google Sheets

  • Buat borang (Google Forms / JotForm / Tally.so) dengan medan: Nama, Program, Tarikh.
  • Integrasikan borang ke Google Sheets. Pastikan sheet mengandungi 5 kolum asas: A:Nama | B:Program | C:Tarikh | D:(kosong) | E:LinkSijil
  • Rename sheet kepada Sheet1 atau sesuaikan dalam kod Apps Script.
3

Generate Sijil & Dapatkan Link

  • Buka Extensions → Apps Script dalam Google Sheets.
  • Padam kod sedia ada & copy-paste skrip di bawah.
  • Tetapkan Trigger (On form submit / manual). Tekan Run — setiap peserta akan dapat PDF + link di kolum E.
Hasil: Setiap baris menerima pautan PDF awam.
Apps Script: janaSijilAutomatik() Google Apps Script (ES5/ES6)
function janaSijilAutomatik() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Sheet1'); // Pastikan nama sheet betul
  const data = sheet.getDataRange().getValues();
  
  // ⚠️ GANTIKAN ID di bawah dengan ID folder & template DOC anda ⚠️
  const DOC_ID = '1Iy4XCmRz4y7K7MUyvlhOAaUEIGT1taq87KgRh4OEvxs'; // ID Google Docs template
  const FOLDER_ID = '1VQaMPsO-QJRN99pWcQu9rGucX2ajI0Zg';        // ID folder Google Drive
  
  const template = DriveApp.getFileById(DOC_ID);
  const folderPenyimpanan = DriveApp.getFolderById(FOLDER_ID);

  // Mula baca data dari baris ke-2 (baris 1 adalah tajuk)
  for (let i = 1; i < data.length; i++) {
    let [id, nama, program, tarikh, linkSijil] = data[i];

    // Kalau dah ada link, skip (tak nak generate dua kali)
    if (linkSijil !== "") continue;

    // 1. Buat salinan template Docs
    let salinan = template.makeCopy(`Sijil - ${nama}`, folderPenyimpanan);
    let docSalinan = DocumentApp.openById(salinan.getId());
    let isiDoc = docSalinan.getBody();

    // 2. Tukar placeholder {{Nama}}, {{Program}}, {{Tarikh}} dengan data sebenar
    isiDoc.replaceText('{{Nama}}', nama);
    isiDoc.replaceText('{{Program}}', program);
    isiDoc.replaceText('{{Tarikh}}', tarikh);
    docSalinan.saveAndClose();

    // 3. Tukar Google Docs ke PDF
    let pdf = salinan.getAs('application/pdf');
    let failPDF = folderPenyimpanan.createFile(pdf);
    failPDF.setName(`Sijil_${nama}.pdf`);
    
    // 4. Padam salinan Google Docs (simpan PDF sahaja)
    salinan.setTrashed(true);

    // 5. Masukkan link PDF ke dalam Google Sheets (kolum E)
    sheet.getRange(i + 1, 5).setValue(failPDF.getUrl());
    
    // 6. Set kebenaran supaya sesiapa dengan link boleh view (untuk JotForm/Tally)
    failPDF.setSharing(DriveApp.Access.ANYONE_WITH_LINK, DriveApp.Permission.VIEW);
  }
  
  SpreadsheetApp.getUi().alert('✅ Selesai! Semua sijil PDF telah dijana & pautan dimasukkan ke ruangan LinkSijil.');
}
Petua: Tukar Sheet1 jika nama sheet berlainan. Pastikan kebenaran Apps Script untuk akses Drive & Docs diberikan.

📌 Panduan Lengkap: Dari Borang ke PDF Bulk

1️⃣

Persediaan Google Drive & Template

  • Buka Google Drive → Baru → Folder (nama: "Output Sijil Bulk"). Dapatkan ID folder dari URL: drive.google.com/drive/folders/INIfolderID
  • Buat Google Docs sebagai template. Tulis teks sijil, masukkan placeholder seperti {{Nama}}, {{Program}}, {{Tarikh}}. Jangan lupa logo & reka bentuk.
  • Dapatkan ID Docs template dari URL dokumen: docs.google.com/document/d/INIadalahIDdoc/edit
2️⃣

Sediakan Borang & Google Sheets (Integrasi)

  • Google Forms: Buat borang baru → Klik Responses → Google Sheets → Pilih spreadsheet baru. Nama sheet akan menjadi "Sheet1".
  • JotForm: Dalam Settings → Integrations → Pilih Google Sheets → Sambung & pilih spreadsheet yang sama. Data akan ditambah ke baris baru secara automatik.
  • Tally.so: Guna Zapier/Make.com: Trigger "New Submission" → Action "Google Sheets - Add Row". Map medan (Nama, Program, Tarikh) ke kolum A,B,C.
  • Pastikan sheet mengandungi 5 kolum dengan susunan: A:Nama | B:Program | C:Tarikh | D:(biarkan kosong) | E:LinkSijil. Apps Script akan mengisi kolum E.
3️⃣

Pasang Apps Script & Generate Sijil

  • Buka Google Sheets (yang terintegrasi borang tadi) → Extensions → Apps Script.
  • Tampal skrip janaSijilAutomatik() yang diberikan di atas. Gantikan DOC_ID & FOLDER_ID dengan ID anda.
  • Klik Run ▶️ → Beri kebenaran (Allow akses ke Drive, Docs, Sheets).
  • Skrip akan membaca semua baris, menghasilkan PDF untuk setiap peserta, dan menulis pautan PDF di kolum E. Jika ada baris baru daripada borang, anda boleh set Trigger automatik: Klik ikon jam di Apps Script → Add Trigger → pilih fungsi janaSijilAutomatik → Event: "On form submit" atau "Time-driven".
  • ✅ Keputusan: Setiap peserta mempunyai pautan sijil PDF dalam sheet. Boleh gunakan pautan itu untuk hantar e-mel atau papar dalam JotForm autoresponder.
Hasil Akhir: Google Sheets kini mengandungi link PDF unik untuk setiap peserta. Anda boleh guna data ini untuk penghantaran e-mel (Gmail/Apps Script MailApp) atau integrasi dengan platform lain.

Cara Hantar Link Sijil ke Peserta

Selepas Apps Script menjana PDF, anda boleh:

  • Guna MailApp.sendEmail() dalam skrip yang sama untuk e-mel automatik.
  • Guna AutoCrat atau JotForm Autoresponder dengan merge tag {{LinkSijil}}.
  • Eksport senarai link dan gunakan platform seperti Mailchimp / WhatsApp Business API.
Tip: Setelah PDF dijana, peserta boleh mengakses sijil mereka melalui pautan awam (tidak perlu log masuk).

Penyelesaian Masalah & Nota Penting

Kebenaran Apps Script: Pastikan semasa pertama kali run, anda izinkan akses ke Google Drive & Dokumen.
Placeholder sensitif: Gunakan {{Nama}} sama persis dengan di template Docs.
Had harian: Apps Script boleh generate ratusan PDF, tetapi ada had kuota (biasanya mencukupi untuk kebanyakan program).
Trigger auto: Untuk setiap borang baru, anda boleh set "onFormSubmit" trigger supaya skrip jalan sendiri.
