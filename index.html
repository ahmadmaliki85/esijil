Terokai Ekosistem Teknologi E-Sijil

Alat bertaraf premium yang menyokong reka bentuk visual, enjin automasi fail, serta borang online berintegrasi tinggi.

Adobe Photoshop

Desktop Only

Pilihan terulung profesional untuk template berkualiti cetakan tinggi. Sangat sesuai untuk reka bentuk grafik yang kompleks namun tidak mempunyai keupayaan automasi dinamik secara langsung (memerlukan eksport manual).

Tiada Automasi Terus Kemahiran: Tinggi

Canva

Web-Based

Terkenal dengan ribuan template sedia ada yang memukau dan mesra pengguna. Memudahkan pereka menyunting sijil secara digital. Mempunyai integrasi terhad melalui Zapier atau Make.com untuk pengeluaran bulk.

Automasi Terhad Kemahiran: Rendah

PowerPoint

MS Office

Mudah untuk reka letak sijil standard. Boleh dihubungkan dengan Microsoft Excel menggunakan ciri tradisional Mail Merge untuk cetakan kelompok, tetapi kurang mesra penyegerakan ke storan awan digital berbanding Google Slides.

Mail Merge Manual Kemahiran: Rendah

Google Docs

Workspace

Sesuai untuk sijil berbentuk surat atau berasaskan teks formal. Sangat serasi dengan Google Sheets dan add-on seperti AutoCrat untuk automasi, namun terhad dari sudut reka bentuk grafik latar belakang yang kompleks.

Serasi AutoCrat Kemahiran: Rendah
Pilihan Terbaik

Google Slides

Native Auto

⭐ PILIHAN TERUTAMA & TERBAIK untuk automasi bertaraf korporat. Sangat fleksibel untuk reka bentuk template sijil dan menyokong integrasi 100% native dengan AutoCrat serta Google Apps Script melalui pemetaan tag placeholder.

Placeholder Tag:
<<Nama Peserta>> {{Nama}}
Native Integration + Dynamic Placeholders Kemahiran: Rendah

Google AutoCrat

Sheets Add-on

Add-on yang dipasang terus pada Google Sheets. Berfungsi memetakan data sel daripada lajur sheet ke dalam placeholder template (Slides/Docs), menjana PDF secara automatik ke dalam Google Drive, dan menghantarnya melalui Gmail. Sangat sesuai untuk pengguna tanpa kemahiran coding.

Kaedah: Mengesan trigger "On Form Submit" atau secara berjadual.
Percuma & Tanpa Kod Kemahiran: Rendah

Google Apps Script

JavaScript Cloud

Enjin pengaturcaraan awan milik Google Workspace. Memberikan kawalan mutlak kepada pembangun untuk memprogramkan logik automasi tersuai (custom JS), membuat salinan fail, menukar slaid kepada dokumen PDF secara dinamik, menulis URL pautan sijil terus ke sheet, and menghantar e-mel dengan fail kepilan sijil secara serentak.

Kebebasan Logik Penuh Kemahiran: Tinggi

Make.com

No-Code Platform

Platform automasi visual (no-code) yang sangat popular di peringkat global. Membolehkan sambungan visual dengan menyeret dan melepaskan modul integrasi (seperti: Tally ➔ Google Sheets ➔ Google Drive ➔ Gmail) dalam satu kanvas visual yang dinamik dan mudah dipantau.

Workflow Visual Sleek Kemahiran: Sederhana

Certifier.io

SaaS Khusus

Platform SaaS premium yang dibina khas untuk menguruskan kitaran sijil digital secara hujung-ke-hujung. Dilengkapi dengan pereka sijil, penghantaran kelompok, dan sistem kod pengesahan QR bersepadu yang membolehkan penerima mengesahkan keaslian sijil mereka.

Pengesahan QR Terbina Kemahiran: Rendah

Google Forms

Integrasi native paling mudah dan 100% percuma. Mengalirkan data peserta terus to Google Sheets secara terus tanpa memerlukan persediaan pihak ketiga. Walau bagaimanapun, mempunyai had yang ketara dari sudut fleksibiliti reka bentuk borang serta ketiadaan logik bersyarat yang kompleks.

Integrasi Native Terus Kemahiran: Rendah

JotForm

Borang kaya dengan fungsi lanjutan seperti tandatangan digital penerima, pembayaran terbina, dan ruangan muat naik fail. Mempunyai integrasi terus ke Google Sheets, namun pelan percuma dihadkan kepada 100 penghantaran borang sahaja sebulan.

Had Pelan Percuma Kemahiran: Rendah

Tally.so

Borang paling moden dengan antara muka minimalis gaya Notion. Pilihan premium & estetik yang memberikan 99% fungsi secara percuma sepenuhnya. Walau bagaimanapun, penyambungan data ke Google Sheets memerlukan Zapier atau Make.com (satu langkah tambahan).

Pilihan Estetik Terbaik Kemahiran: Sederhana

Typeform

Platform borang premium terkemuka dengan gaya perbualan "satu soalan pada satu masa" yang meningkatkan kadar interaksi peserta. Sangat cantik tetapi berbayar tinggi untuk menyahkunci integrasi API pihak ketiga secara meluas.

Pelan Premium Mahal Kemahiran: Sederhana

Kod Pakej

Klip Kod Apps Script Bersedia Untuk Digunakan

Pasangkan skrip automasi ini dalam tetingkap Extensions ➔ Apps Script di fail Google Sheets anda. Gantikan ID dokumen dengan kepunyaan anda sendiri.

janaSijil.js Google Apps Script 
function janaSijilAutomatik() {
  // 1. Kenal pasti helaian Sheet1 yang mengandungi rekod peserta
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Sheet1');
  const data = sheet.getDataRange().getValues();
  
  // ⚠️ GANTIKAN DENGAN ID GOOGLE SLIDES TEMPLATE & FOLDER DRIVE ANDA
  const SLIDES_ID = '1AbCdEfGhIjKlMnOpQrStUvWxYz1234567890abcdefg'; 
  const FOLDER_ID = '1aBcDeFgHiJkLmNoPqRsTuVwXyZ098765';         
  
  const template = DriveApp.getFileById(SLIDES_ID);
  const folderPenyimpanan = DriveApp.getFolderById(FOLDER_ID);

  // Mula pembacaan baris dari baris ke-2 (Baris 1 ialah tajuk lajur)
  for (let i = 1; i < data.length; i++) {
    let [id, nama, program, tarikh, linkSijil] = data[i];

    // Jika lajur pautan sudah terisi, langkau (elak penjanaan berganda)
    if (linkSijil !== "") continue;

    // A. Buat pendua slaid template baharu dalam folder storan
    let salinan = template.makeCopy(`Sijil - ${nama}`, folderPenyimpanan);
    let slideSalinan = SlidesApp.openById(salinan.getId());
    let presentation = slideSalinan.getSlides()[0]; // Slide pertama

    // B. Gantikan placeholder tag {{Nama}}, {{Program}}, {{Tarikh}}
    presentation.replaceAllText('{{Nama}}', nama);
    presentation.replaceAllText('{{Program}}', program);
    presentation.replaceAllText('{{Tarikh}}', tarikh);
    slideSalinan.saveAndClose();

    // C. Tukarkan pendua Google Slides tadi kepada salinan fail PDF
    let pdf = salinan.getAs('application/pdf');
    let failPDF = folderPenyimpanan.createFile(pdf);
    failPDF.setName(`Sijil_${nama}.pdf`);
    
    // Padam slaid pendua asal (Kekalkan fail PDF sahaja)
    salinan.setTrashed(true);

    // D. Rekodkan pautan fail PDF awam ke lajur E (lajur ke-5)
    sheet.getRange(i + 1, 5).setValue(failPDF.getUrl());
    
    // Set kebenaran perkongsian fail kepada 'Siapa dengan Pautan boleh view'
    failPDF.setSharing(DriveApp.Access.ANYONE_WITH_LINK, DriveApp.Permission.VIEW);
  }
}

Gabungan Sistem Terbaik (Pilihan Kami)

  • Reka Bentuk: Gunakan Google Slides. Sediakan slaid bersaiz A4 melintang dengan kotak teks tag placeholders bercerun kemas.
  • Kutipan Borang: Gunakan Google Forms (Paling mudah) atau Tally.so (Antara muka visual Notion estetik + percuma).
  • Enjin Penjana: Gunakan AutoCrat untuk mula membina sistem dalam 15 minit tanpa kod, atau Apps Script untuk automasi bertulis.
