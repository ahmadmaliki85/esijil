PDF & Emel Automatik

Dua versi kod Google Apps Script siap guna. Versi pertama untuk menjana PDF sijil dan menghantarnya terus ke e-mel peserta bengkel, manakala versi kedua dikhususkan untuk menjana PDF dan menyimpannya di Google Drive sahaja.

janaSijilDanEmel.js Google Apps Script API 
function janaSijilDanEmelPeserta() {
  // 1. Sambungkan helaian Sheet1 dalam spreadsheet aktif
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Sheet1');
  const data = sheet.getDataRange().getValues();
  
  // ⚠️ GANTIKAN DENGAN ID TEMPLATE SLIDES & FOLDER DRIVE ANDA YANG SEBENAR
  const SLIDES_ID = 'MASUKKAN_ID_TEMPLATE_SLIDES_ANDA'; 
  const FOLDER_ID = 'MASUKKAN_ID_FOLDER_OUTPUT_DRIVE_ANDA';   
  
  const template = DriveApp.getFileById(SLIDES_ID);
  const folderPenyimpanan = DriveApp.getFolderById(FOLDER_ID);

  // Mula baca baris dari baris ke-2 (Baris 1 ialah tajuk lajur)
  // Susunan Lajur: A:Nama | B:Program | C:Tarikh | D:E-mel | E:Link Sijil
  for (let i = 1; i < data.length; i++) {
    let [nama, program, tarikh, emel, linkSijil] = data[i];

    // Langkau jika lajur E (Link Sijil) sudah terisi atau E-mel tiada
    if (linkSijil !== "") continue;
    if (emel === "") continue;

    try {
      // 2. Buat pendua slaid template baharu dalam folder storan
      let salinan = template.makeCopy(`Sijil - ${nama}`, folderPenyimpanan);
      let slideSalinan = SlidesApp.openById(salinan.getId());
      let presentation = slideSalinan.getSlides()[0]; // Slaid pertama

      // 3. Gantikan Placeholder tag {{Nama}}, {{Program}}, {{Tarikh}}
      presentation.replaceAllText('{{Nama}}', nama);
      presentation.replaceAllText('{{Program}}', program);
      presentation.replaceAllText('{{Tarikh}}', tarikh);
      slideSalinan.saveAndClose();

      // 4. Convert fail slaid pendua tadi kepada PDF
      let pdf = salinan.getAs('application/pdf');
      let failPDF = folderPenyimpanan.createFile(pdf);
      failPDF.setName(`Sijil_${nama}.pdf`);
      
      // Padam slaid pendua asal (Kekalkan fail PDF sahaja)
      salinan.setTrashed(true);

      // Kebenaran perkongsian fail awam (Supaya pautan dalam e-mel boleh diakses)
      failPDF.setSharing(DriveApp.Access.ANYONE_WITH_LINK, DriveApp.Permission.VIEW);
      let pdfUrl = failPDF.getUrl();

      // 5. Tulis URL sijil PDF pada lajur E (lajur ke-5) Google Sheets
      sheet.getRange(i + 1, 5).setValue(pdfUrl);

      // 6. HANTAR E-MEL BERSAMA FAIL KEPILAN PDF
      let tajukEmel = `Sijil Penghargaan: ${program}`;
      let mesejEmel = `Salam sejahtera ${nama},\n\n` +
                       `Tahniah! Anda telah berjaya menyelesaikan program "${program}" pada ${tarikh}.\n\n` +
                       `Bersama-sama e-mel ini dilampirkan sijil penghargaan anda dalam format PDF.\n\n` +
                       `Anda juga boleh mengakses salinan sijil anda pada bila-bila masa melalui pautan Drive berikut:\n${pdfUrl}\n\n` +
                       `Sekian, terima kasih.`;

      MailApp.sendEmail({
        to: emel,
        subject: tajukEmel,
        body: mesejEmel,
        attachments: [failPDF.getAs(MimeType.PDF)]
      });

      console.log(`[SUKSES] Sijil dijana & e-mel dihantar kepada: ${emel}`);
      
    } catch (err) {
      console.error(`[RALAT] Ralat pada baris ${i + 1}: ${err.message}`);
    }
  }
}

Struktur Lajur Sheets

Struktur Helaian Data Google Sheets (Wajib)

Untuk memastikan kod Apps Script berjalan tanpa sebarang ralat, anda wajib menyusun struktur tajuk lajur Google Sheets bermula daripada Lajur A hingga Lajur E seperti berikut:

Lajur Nama Tajuk Keterangan
Kolum A Nama Nama penuh rasmi peserta (Siri)
Kolum B Program Nama acara/bengkel akademik
Kolum C Tarikh Tarikh tamat pelaksanaan program
Kolum D E-mel Alamat e-mel peserta (Wajib untuk Skrip A)
Kolum E LinkSijil Output URL ditulis automatik oleh skrip

Had & Had Kuota Google Apps Script

Apabila melaksanakan sistem automasi e-sijil berkelompok, anda tertakluk kepada kekangan perkakasan dan kuota API awan Google Workspace:

Had Hantar E-mel (Gmail) 100 E-mel / Hari

Akaun Gmail percuma dihadkan kepada 100 sehari. Akaun Google Workspace berbayar dihadkan kepada 1,500 sehari.

Execution Timeout 6 Minit / Run

Setiap proses larian skrip dihadkan kepada maksimum 6 minit sahaja. Jika data terlalu banyak, laksanakan secara berperingkat.

Trigger Limit 20 Triggers / Akaun

Had maksimum 20 trigger aktif bagi setiap akaun pembangun Google Apps Script.

Pencegahan Double-Send If (linkSijil !== "")

Logik perlindungan sedia ada dalam skrip bertujuan mengelakkan penghantaran e-mel dan fail berganda kepada peserta sama.

Setup Manual

Cara Pemasangan Kod di Google Sheets

Ikuti 4 langkah ringkas untuk memasang, memberi kebenaran keselamatan, dan melancarkan skrip automasi anda.

01

Buka Apps Script

Buka Google Sheets yang mengandungi senarai data peserta. Klik pada menu Extensions ➔ Apps Script di bar menu atas.

Tetingkap Kod Dibuka
02

Tampal Kod & Ganti ID

Padam kod laluan sedia ada di editor. Tampal Skrip A atau B yang disalin dari portal ini. Gantikan ID Template Slides & ID Folder Output Drive dengan milik anda sendiri.

Tekan Ctrl + S untuk simpan
03

Beri Kebenaran (OAuth)

Klik butang Run ▶️. Google akan memaparkan tetingkap keselamatan. Klik Advanced ➔ Go to Untitled (unsafe) ➔ Allow untuk meluluskan akses Google Drive, Slides & Gmail.

Kelulusan Keselamatan Sekali sahaja
04

Add Trigger Automatik

Untuk larian automatik setiap kali peserta mengisi borang online, klik ikon jam (Triggers) di menu kiri Apps Script. Klik Add Trigger ➔ Event Type: "On form submit".
